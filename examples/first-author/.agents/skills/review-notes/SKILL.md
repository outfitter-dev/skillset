# Review Notes

Use this skill when a notes document needs to become a short decision log.

## Workflow

- Identify concrete decisions and preserve the wording that matters.
- Pull out follow-ups with an owner when one is named.
- Separate unresolved risks from agreed next steps.
- End with the smallest useful summary someone can act on.
